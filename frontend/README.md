# Frontend (React + Vite)

## Setup
```
cd frontend
npm install
npm run dev
```

## Structure
- `src/App.jsx` — routes and layout
- `src/pages/` — Landing, Dashboard, Portfolio
- `src/components/` — Idea cards, market/competitor panels, scores, partners
- `src/services/api.js` — Axios client
- `src/context/AuthContext.jsx` — simple auth token handling

Update `api.js` baseURL if the backend host/port changes.
